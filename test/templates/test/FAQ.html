Frequently Asked Questions

Log in to your account and select the option "Create a test". Come up with a test title, choose its category, add questions, answer options, and specify the correct answers. Set the test completion time and save it.
Select an available test from the list on the main page and start taking it. Answer the questions by choosing one or multiple answers from the options provided. Test results will be displayed after its completion.
You can edit the tests you have created on the "My tests" page, as well as view statistics on completed tests on the "Passed tests" page.
You can find a test on the main page among the list of tests, as well as use the search bar in the navigation panel. Search is done based on a keyword in the test title. Search can also be done by test categories by clicking on one of the categories in the list. Test categories can be viewed in the dropdown list on the navigation panel, called "Categories".
If you have problems or questions, contact our manager on Telegram (@staishal) or send us an email (alba.shaykhullina@mail.ru). We are always ready to help you!
